Python OOP Calculator (Tkinter)

This project is a simple graphical calculator built using Python. 
It uses Tkinter for the user interface and follows Object-Oriented Programming (OOP) principles. 
The calculator performs basic arithmetic operations and includes features such as history tracking 
and a factory reset system that clears both the interface and stored data.

Features
Basic arithmetic operations:
  - Addition (+)
  - Subtraction (-)
  - Multiplication (×)
  - Division (/)
  - Exponent (^)

- Graphical User Interface (GUI) using Tkinter  
- Stores calculation history in text files  
- Displays last saved calculation  
- Factory reset functionality:
  - Clears calculator input
  - Clears display
  - Deletes saved history and previous calculation files  

Uses Object-Oriented Programming (OOP):
  - Inheritance (CalculatorState → CalGui)
  - Encapsulation of logic and features
